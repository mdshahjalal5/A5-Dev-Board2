Question-1: What are the different ways to select an element in the DOM?

DOM-এ একটি এলিমেন্ট সিলেক্ট করার বিভিন্ন পদ্ধতি নিচে তুলে ধরা হলো:

1. `document.getElementById()`: একটি নির্দিষ্ট `id` সহ এলিমেন্ট সিলেক্ট করে।

2. `document.getElementsByClassName()`: একটি নির্দিষ্ট ক্লাসের সব এলিমেন্ট সিলেক্ট করে।

3. `document.getElementsByTagName()`: নির্দিষ্ট ট্যাগ নামের সব এলিমেন্ট সিলেক্ট করে।

4. `document.querySelector()`: প্রথম ম্যাচ করা এলিমেন্টটি সিলেক্ট করে।

5. `document.querySelectorAll()`: সব ম্যাচ করা এলিমেন্টগুলো সিলেক্ট করে।

6. `getElementById()` এবং `getElementsByClassName()`: প্রথম এবং সব ম্যাচ করা এলিমেন্ট সিলেক্ট করে।

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText, and textContent is as follows:

1. `innerHTML`: Provides and can modify an element's complete content including HTML tags. It parses HTML and can execute scripts when content is set.

2. `innerText`: Provides only human-readable text as displayed by CSS output. It doesn't show hidden elements and doesn't parse HTML tags.

3. `textContent`: Provides all text content of an element, even content hidden by CSS. It doesn't parse HTML tags and doesn't execute scripts.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where instead of attaching an event listener to each individual element, we attach a single event listener to a parent element that will fire for all descendants matching a specified selector. This technique leverages event bubbling, where events triggered on a nested element "bubble up" through its ancestors in the DOM tree.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM where an event triggered on a nested element propagates upward through the DOM hierarchy to its parent elements.

When an event occurs on an element (like a click on a button), the event first runs on that element, then on its parent, then on its grandparent, and so on through the entire hierarchy until it reaches the root document object.

Key aspects of event bubbling:

Propagation order: From the target element upward to the document root.

Event phases: The DOM event flow has three phases:

Capture phase: Event travels down to the element Target phase: Event reaches the target element Bubbling phase: Event bubbles up from the element

Stopping propagation: You can prevent an event from continuing to bubble by using event.stopPropagation().

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element in JavaScript, we use the document.createElement() method. This generates a new element in memory but does not place it on the page yet. To add the created element to the page, we use appendChild() or append(), which attach the element to a parent element, making it visible in the document. To modify an element, we use properties like textContent, innerHTML, or setAttribute() to change its content, structure, or attributes. To remove an element, we use remove() to delete it directly or removeChild() to remove it through its parent.
